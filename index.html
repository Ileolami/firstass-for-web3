HTML TAGS

    HTML TAGS are:

  1. (!--...--) Defines a comment
  2. (!DOCTYPE) Defines the document type
  3. (a) Defines a hyperlink
  4. (abbr) Defines an abbreviation or an acronym
  5. (sub) Defines subscripted text
  6. (summary) Defines a visible heading for a (details)element
  7. (dl) Defines a description list
  8. (sub) Defines subscripted text
  9. (sup) Defines superscripted text
  10. (template) Defines a container for content that should be hidden when the page loads
  11. (time) Defines a specific time (or datetime)
